# Minecraft-Changelog-Generator
Tool for autocreating modpack changelog
Compatible only with Modrinth (currently)

**Instalation:**  
`npm install minecraft-changelog-generator`

**Aliases:**
* auto-changelog
* ac
* changelog-generator
* cg

**Example:**
```bash
cg path/to/old.mrpack path/to/new.mrpack
```
